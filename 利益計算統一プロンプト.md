# Claude Code 実装指示: 日別ページの利益計算をダッシュボードと統一する

## 問題

ダッシュボード（`src/app/page.tsx`）と商品分析ページでは **商品ID別** に広告費を集計して利益を計算しているが、日別ページ（`src/app/(dashboard)/daily/page.tsx`）では **日付別** に広告費を集計している。

この集計方法の違いにより、利益の合計値に差額（約¥8,913）が発生している。

- ダッシュボード: `getProductSalesSummary()` → 商品ごとに `ad_spend` を紐付け → 合計 ¥1,148,223
- 日別ページ: `getDailyAdSpendByDate()` → 日付ごとに `ad_spend` を合算 → 合計 ¥1,157,136

原因: `daily_advertising` テーブルに `product_id` が NULL のレコードや、`daily_sales` に存在しない `product_id` のレコードがある可能性がある。日付ベースの集計ではこれらも含まれるが、商品IDベースでは含まれない。

## やること

日別ページの広告費集計を **商品IDベース** に統一する。具体的には `getDailyAdSpendByDate()` の代わりに、`getAdSummary()` の日別版を作成して使う。

**変更ファイル:**
1. `src/lib/api/advertising.ts` — 新関数追加
2. `src/app/(dashboard)/daily/page.tsx` — 広告費取得を新関数に変更

## ステップ1: advertising.ts に新関数を追加

`src/lib/api/advertising.ts` に以下の関数を追加:

```typescript
/**
 * 日別の広告費を取得（商品IDベースで集計し、日付でグループ化）
 * product_id が daily_sales に存在する商品のみ集計するため、
 * getProductSalesSummary() と同じ集計ロジックになる。
 */
export async function getDailyAdSpendByDateProductBased(params: {
  startDate?: string;
  endDate?: string;
}) {
  // 1. daily_advertising から全データ取得
  let adQuery = supabase
    .from("daily_advertising")
    .select("date, product_id, ad_spend");

  if (params.startDate) adQuery = adQuery.gte("date", params.startDate);
  if (params.endDate) adQuery = adQuery.lte("date", params.endDate);

  const { data: adData, error: adError } = await adQuery;
  if (adError) { console.warn("getDailyAdSpendByDateProductBased ad error:", adError); return {}; }

  // 2. daily_sales から期間内の product_id 一覧を取得
  let salesQuery = supabase
    .from("daily_sales")
    .select("product_id");

  if (params.startDate) salesQuery = salesQuery.gte("date", params.startDate);
  if (params.endDate) salesQuery = salesQuery.lte("date", params.endDate);

  const { data: salesData, error: salesError } = await salesQuery;
  if (salesError) { console.warn("getDailyAdSpendByDateProductBased sales error:", salesError); return {}; }

  // 有効な product_id のセットを作成
  const validProductIds = new Set(
    (salesData || []).map((s: any) => s.product_id).filter(Boolean)
  );

  // 3. 有効な product_id の広告費のみ日付でグループ化
  const byDate: Record<string, number> = {};
  for (const row of adData || []) {
    if (row.product_id && validProductIds.has(row.product_id)) {
      byDate[row.date] = (byDate[row.date] || 0) + (row.ad_spend || 0);
    }
  }
  return byDate;
}
```

## ステップ2: daily/page.tsx の広告費取得を変更

`src/app/(dashboard)/daily/page.tsx` で:

### 2-1. import を変更

```typescript
// 変更前
import { getDailyAdSpendByDate } from "@/lib/api/advertising";

// 変更後
import { getDailyAdSpendByDateProductBased } from "@/lib/api/advertising";
```

### 2-2. データ取得部分を変更

`getDailyAdSpendByDate` を呼んでいる箇所（TanStack Query の `queryFn` 内）を `getDailyAdSpendByDateProductBased` に置き換え:

```typescript
// 変更前
const adSpendByDate = await getDailyAdSpendByDate({ startDate, endDate });

// 変更後
const adSpendByDate = await getDailyAdSpendByDateProductBased({ startDate, endDate });
```

他の部分（`adSpendByDate[date]` の参照箇所）は返り値の型が同じなので変更不要。
