## 実装手順

### 1. product_group を使ったグループ分けロジックを追加

現在の `combinedProductRanking` 変数の直後（120行目あたり）に以下を追加:

```typescript
// 商品グループ一覧を抽出（product_group が null の場合は "その他"）
const productGroups = (() => {
  const groupSet = new Set<string>();
  for (const p of combinedProductRanking) {
    // Amazon: p.product_group, 楽天: p.product_group（両方対応）
    groupSet.add(p.group || "その他");
  }
  return ["全商品", ...Array.from(groupSet).sort()];
})();
```

### 2. combinedProductRanking にグループ情報を追加

現在の `combinedProductRanking` 定義（104〜120行目）を以下に修正:

```typescript
// Amazon + 楽天 統合商品ランキング（グループ情報付き）
const combinedProductRanking = [
  ...(productSummary as any[]).map((p: any) => ({
    name: p.product?.name || "不明",
    channel: "Amazon" as const,
    group: p.product?.product_group || "その他",
    total_sales: p.total_sales,
    total_orders: p.total_orders,
    net_profit: p.net_profit || 0,
  })),
  ...(rakutenProductSummary as any[]).map((p: any) => ({
    name: p.product?.name || "不明",
    channel: "楽天" as const,
    group: p.product?.product_group || "その他",
    total_sales: p.total_sales,
    total_orders: p.total_orders,
    net_profit: p.net_profit || 0,
  })),
].sort((a, b) => b.total_sales - a.total_sales);
```

**ポイント:** `group: p.product?.product_group || "その他"` を追加。Amazon の `products` テーブルも楽天の `rakuten_products` テーブルも両方 `product_group` フィールドを持っている。

### 3. タブの state を追加

`const [period, setPeriod] = useState("30days");` の直後に追加:

```typescript
const [activeGroup, setActiveGroup] = useState("全商品");
```

### 4. グループ一覧と絞り込みデータの計算を追加

`productGroups` の直後に以下を追加:

```typescript
// アクティブグループで絞り込んだ商品一覧
const filteredProducts = activeGroup === "全商品"
  ? combinedProductRanking
  : combinedProductRanking.filter(p => p.group === activeGroup);

// グループ合計
const groupTotalSales = filteredProducts.reduce((sum, p) => sum + p.total_sales, 0);
const groupTotalOrders = filteredProducts.reduce((sum, p) => sum + p.total_orders, 0);
const groupTotalProfit = filteredProducts.reduce((sum, p) => sum + p.net_profit, 0);
const groupProfitRate = groupTotalSales > 0 ? (groupTotalProfit / groupTotalSales) * 100 : 0;
```

## 技術メモ

### product_group の取得元

| チャネル | テーブル | フィールド | API関数 |
|---------|---------|-----------|--------|
| Amazon | products | product_group | getProductSalesSummary → `p.product.product_group` |
| 楽天 | rakuten_products | product_group | getRakutenProductSalesSummary → `p.product.product_group` |

両方とも `product_group` フィールドを持っており、API関数で `select("*, product:products(*)")` / `select("*, rakuten_product:rakuten_products(*)")` として関連テーブルの全フィールドを取得しているので、追加のAPI変更は不要。
